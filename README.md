# RANA Website Designers

A responsive portfolio landing page for **Rana Website Designers**, showcasing web design, UI/UX, web development, and IT/computer support services.

## Overview

This project is a single-page HTML portfolio website built using:

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Remixicon icon library
- Remote imagery from a placeholder image service

The page includes:

- Hero section with call-to-action buttons
- Portfolio showcase with project cards
- Services section with feature cards
- About section with skill progress bars
- Contact form with validation and mailto support
- Mobile menu and smooth scrolling navigation
- Modals for hire recommendations and project details
- Back-to-top button and sticky header

## Getting Started

### Option 1: Open in browser

Simply open `index.html` in your web browser.

### Option 2: Run a local server

A local web server is recommended for the best experience.

From the project directory:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

### Option 3: Use VS Code Live Server

If you have the VS Code Live Server extension installed, open the folder and click **Go Live**.
